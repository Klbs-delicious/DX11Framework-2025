# DX11Framework-2025

DirectX11 / C++ を用いた就職作品制作リポジトリです。  
アクションゲーム制作を進めながら、ゲーム実装用フレームワークも並行して構築しています。

現在は制作途中のため、このリポジトリは主にコード・設計確認用です。

## 使用技術
- C++20
- DirectX11
- HLSL

## 補足
容量の大きいモデル・一部データは除外しています。
そのため、clone後にそのまま実行できる状態ではありません。

## 主な実装
- コンポーネントベースのゲームオブジェクト管理
- TimeScaleによるスロー演出制御
- ポストエフェクト処理
- アニメーション遷移管理
- ジャスト回避判定
- イベントシステム
- デバッグ機能
